# Membuat Sistem Machine Learning

Selamat datang di repository proyek akhir saya! Proyek ini mencakup beberapa tahapan penting dalam membangun sistem machine learning secara utuh, termasuk eksperimen dataset, pembangunan model, pembuatan CI/CD pipeline, dan sistem monitoring & logging.

## Deskripsi Proyek
Proyek ini bertujuan untuk membangun sistem machine learning yang lengkap dengan berbagai komponen penting, mulai dari eksperimen terhadap dataset pelatihan hingga implementasi sistem monitoring dan logging yang efektif. Berikut adalah beberapa komponen utama yang telah dibangun:

- **Eksperimen terhadap Dataset Pelatihan**: Pengolahan dataset dilakukan melalui preprocessing dan analisis eksploratif untuk mempersiapkan data sebelum digunakan dalam pelatihan model.
- **Pembangunan Model Machine Learning**: Model dipilih dan dilatih menggunakan berbagai teknik, termasuk hyperparameter tuning untuk mengoptimalkan performa model.
- **Workflow CI (Continuous Integration)**: Pipeline otomatis untuk menjalankan eksperimen, pelatihan model, dan pengelolaan artefak menggunakan GitHub Actions.
- **Sistem Monitoring dan Logging**: Penggunaan Grafana dan Prometheus untuk memantau kinerja model, dengan implementasi alerting untuk memantau perubahan performa model.

## Fitur
- **Preprocessing Dataset**: Memuat berkas otomatisasi preprocessing untuk memudahkan replikasi eksperimen.
- **Hyperparameter Tuning**: Menggunakan pustaka seperti Optuna untuk optimasi model.
- **CI/CD Pipeline**: Mengautomasi proses training dan penyimpanan artefak melalui GitHub Actions.
- **Monitoring & Logging**: Implementasi Prometheus untuk pengumpulan metrik dan Grafana untuk visualisasi data serta alerting.
